Our Specialty

Read below to find an overview of the process taken by Block Drive and why it is a much better option that other alternatives.

Upload

The user picks a file to be uploaded onto the website.

Split

The file uploaded is disassebled into multiple, equally large pieces for upload using the InterPlanetary File System created and the user must pay a transaction fee that is assisted by the Hedera API.

Encryption

The files put on the hardrive is encrypted so that no one without permission is able to access the file. They are then recombined into a single file to be decripted on the hardrive.

File Storage and Sharing

Once the user has paid the microtransaction for storing their material in a hard drive, the price being based on multiple factors such as space taken up and time the space is rented, other users are able to access it once they are given permission and will also have to pay a small fee to access it.

You're stuff, even more your way - Block Drive Features

With a B Drive, you can store your file on one of our many hard drives after paying a minimal microtransacion.

Here, you can save any attachment to your Drive to organize and share them in a single, safe place. You can also create and collaborate with others and share documents and files.
